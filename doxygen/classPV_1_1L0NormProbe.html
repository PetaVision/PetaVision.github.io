Detailed Description

A layer probe for returning the number of elements in its target layer's activity buffer above a certain threshold (often referred to as the L0-norm).

getValueInternal(double, index) is a pure virtual function called by getValue() and getValues(). The index refers to the layer's batch element index.

getValue(t, index) returns the sum of each MPI process's getValueInternal(t, index). getValues(t, values) returns a vector whose kth element is the value that getValue(t, k) returns.
