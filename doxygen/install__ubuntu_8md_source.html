1 Ubuntu Installation
2 ==================================
3 
4 This page provides instructions for installing PetaVision on
5 Ubuntu&nbsp;14.04 (Trusty Tahr), beginning with a clean installation of Ubuntu.
6 You will need administrator privileges to run sudo.
7 
8 Required packages
9 ----------------------------------
10 ~~~~~~~~~~~~~~~~~~~~{.sh}
11 sudo apt-get update
12 sudo apt-get dist-upgrade
13 sudo apt-get install bison cmake cmake-curses-gui flex gcc g++ git openmpi-bin libgdal-dev libopenmpi-dev openmpi-bin
14 ~~~~~~~~~~~~~~~~~~~~
15 
16 Optional packages
17 ----------------------------------
18 ~~~~~~~~~~~~~~~~~~~~{.sh}
19 sudo apt-get install lua5.2 # for using lua scripts to manage parameter files
20 sudo apt-get install octave octave-image # for using octave .m scripts in the analysis folder
21 echo pkg load all >> ~/.octaverc # loads octave image package etc. when starting octave.
22 sudo apt-get install gdb # Line debugger for gcc and g++
23 ~~~~~~~~~~~~~~~~~~~~

25 CUDA and NVIDIA Driver (Optional)
26 ----------------------------------
27 Our GPU acceleration depends on CUDA. While it is possible to build and run
28 PetaVision without CUDA, performance will not be nearly as good.
29 
30 To download CUDA, go to <https://developer.nvidia.com/cuda-downloads>.
31 Click the RUN file for your architecture and your version of Ubuntu. Note the location.
32 
33 The version of the NVIDIA driver installed by CUDA may be behind the most
34 recent driver version released by NVIDIA. If you want to use a more recent
35 driver, you will need to install CUDA first, along with the driver that CUDA
36 installs, and then upgrade the NVIDIA driver.
37 If you want to use the most recent NVIDIA driver, go to
38 <http://www.nvidia.com/Download/index.aspx?lang=en-us> and select your
39 video card. Download the linux RUN file. Note the location.
40 
41 NVIDIA drivers cannot be installed while the X window system is running. Here's how to install the drivers.
42 
43 Enter a virtual console (usually by pressing control-alt-F1 or control-alt-fn-F1).
44 Log into an account with admin privileges.
45 ~~~~~~~~~~~~~~~~~~~~{.sh}
46 sudo service lightdm stop
47 chmod u+x path/to/cuda.run
48 sudo path/to/cuda.run
49 ~~~~~~~~~~~~~~~~~~~~
50 
51 Follow the onscreen instructions. We have seen the warning "The distribution-provided pre-install script failed!" but
52 it does not seem to cause problems if you continue.
53 
54 When it asks if you want to install NVIDIA driver, select yes. Add a symbolic link to `/usr/local/cuda` when asked.
55 
56 
57 If you now want to upgrade the NVIDIA driver to a version downloaded above.
58 ~~~~~~~~~~~~~~~~~~~~{.sh}
59 cd path/to/driver.run
60 chmod u+x path/to/driver.run
61 ./path/to/driver.run
62 ~~~~~~~~~~~~~~~~~~~~
63 
64 Follow the instructions on the screen.
65 When asked to uninstall the existing driver (installed by cuda), select yes.
66 When asked to update your Xorg, select yes. This will use this driver as your GUI display.
67 
68 Run `nvidia-smi` and make sure you see your video card.
69 
70 Reboot.
71 
72 cuDNN (optional)
73 ----------------------------------
74 You can optionally use the cuDNN Deep Neural Network Library, which provides additional
75 speed-up in computing the convolutions we use to compute input to post-synaptic layers.
76 
77 To download cuDNN, go to <https://developer.nvidia.com/cuDNN> and click Download.
78 If you haven't already registered with NVIDIA Developer you will need to do so, and then wait for email confirmation.
79 Once registered, log into NVIDIA Developer and download the cuDNN Library for Linux.
80 (Alternatively, if you have access to NMC's compneuro, grab it from `/nh/compneuro/Data/cuDNN`)
81

87 sudo apt-get dist-upgrade

91 Checking Out and Installing PetaVision
92 ----------------------------------
93 
94 ~~~~~~~~~~~~~~~~~~~~{.sh}
95 cd ${HOME}
96 mkdir workspace
97 cd workspace
98 git clone https://github.com/PetaVision/OpenPV
99 ccmake .
100 ~~~~~~~~~~~~~~~~~~~~
101 
102 CMake options:
103 ~~~~~~~~~~~~~~~~~~~~
104 CMAKE_BUILD_TYPE = Release #Optimiztaion for CPU
105 PV_USE_CUDA = ON #Depending on if you want to use GPUS
106 CUDA_RELEASE = True #Optimization for Cuda
107 CUDNN = True #If you're using CUDNN
108 CUDNN_PATH = /path/to/cudnn/folder #The path to the cuDNN folder you downloaded/copied
109 PV_USE_OPENMP_THREADS = ON #Whether to use OpenMP threads for parallelization
110 ~~~~~~~~~~~~~~~~~~~~
111 
112 If some of these options do not show up on ccmake, fill the ones you can, press c to configure, and look again for variables
113 Press G to generate when avaliable.
114 
115 Running the system tests:
116 
117 ~~~~~~~~~~~~~~~~~~~~{.sh}
118 cd ~/workspace/PVSystemTest
119 make
120 ctest
121 ~~~~~~~~~~~~~~~~~~~~

124 Optional: Anaconda Python
125 ----------------------------------
126 Ubuntu comes with python 2.7. We suggest installing Anaconda Python for analysis tools.
127 - Go to <https://store.continuum.io/cshop/anaconda/>
128 - Enter your email
129 - Click I WANT PYTHON 3.4
130 - Click the linux penguin
131 - Click the download link and save it
132 
133 ~~~~~~~~~~~~~~~~~~~~{.sh}
134 chmod u+x path/to/python.sh
135 ./path/to/python.sh
136 ~~~~~~~~~~~~~~~~~~~~
137 
138 Follow the directions. Make sure to specify yes when it asks you if you want anaconda in your path.
139 
140 ~~~~~~~~~~~~~~~~~~~~{.sh}
141 source ~/.bashrc
142 ~~~~~~~~~~~~~~~~~~~~
143 
144 To ensure it's working, type python and make sure the text splash says `Python 3.4.1 |Anaconda 2.1`
